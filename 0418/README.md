|명령어|의미|

# 9장
## 1. 명령어 스케줄링
- cron: 주기적으로 명령 실행 (crontab 파일 사용)

- crontab -l, -e, -r: 목록 보기, 편집, 삭제
  - 형식: 분 시 일 월 요일 사용자 명령

- at: 특정 시간에 명령을 한 번만 실행
  - at, atq, at -r 등으로 사용

## 2. 디스크 및 아카이브
- df: 디스크 사용량 확인

- du: 파일 또는 디렉터리의 디스크 사용량 확인

- tar: 여러 파일을 하나의 아카이브로 묶기 (압축은 아님)
  - -cvf, -xvf, -tvf 등 옵션 사용

## 3. 파일 압축
- gzip / gunzip: .gz 파일 압축/해제

- compress / uncompress: .Z 확장자 사용

## 4. AWK
- 텍스트 파일을 줄 단위로 읽고, 각 줄을 필드 단위로 나누어 조건에 따라 처리

- 기본 형식: [조건] { 액션 }

- 주요 기능:
  - 필드 처리 ($1, $NF, NF, NR)

- 조건문, 반복문, 내장 함수 사용 가능

- BEGIN / END 블록 사용
