## C언어 기초
- 변수의 뜻: 변하는 수
- 제어문 종류
	- if, else if, else, goto, continue, break 등
	- goto문은 사용하지 않음
- 함수의 리턴문
- 함수 뜻

- 배열: 같은 타입의 변수들로 이루어진 유한 집합
	- 인덱스를 사용
	- for, while문 이용해서 전체 탐색
	- 배열이름에는 시작 주소

- 포인터

## 리눅스
- echo
- cp (copy)
	- 파일을 복사
	- -r 옵션: 전체 복사

- mv (move)
	- 파일 이동
	- 이름 변경

- rmdir
	- 디렉터리 전체 삭제
	- rm -rf: 디렉터리 내에 파일이 있어도 무시하고 전체 삭제 (주의해서 사용)

- ln: 링크
	- 복사와 다르게 원본 파일을 수정하면 링크된 파일들이 다같이 수정됨

## 파일 속성
- 블록 수, 파일 종류, 접근 권한, 링크 수, 소유자 및 그룹, 수정 시간
- 파일 권한
	- chmod 사용
	- u: 유저, g: 그룹, o: 기타
	- ex) chmod u-rw hello.c
